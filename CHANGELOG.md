# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.5.0] - 2026-03-18

### Added
- **메인페이지 재설계**:
  - 오늘 시간별 풍속 예측 차트 (구간별 색상 라인)
  - 오늘 시간별 발전량 바 차트
  - 발전소 위치 Leaflet 위성지도 + WTG#1~#20 실제 좌표 마커
- **하루전예측 발전소 탭**: 24시간 풍속 라인 차트 + 발전량 KPI/시간별 차트
- **하루전예측 터빈별 탭**: WTG#1~#20 개별 발전량+풍속 듀얼 Y축 차트

### Changed
- **탭 이름 변경**: 단기예측→하루전예측, 중기예측→2주예측, 장기예측→3개월예측
- **개요 탭**: 개요→서남해해상풍력
- **하루전예측**: 3일(72시간)→1일(24시간)으로 변경
- **하루전예측 터빈별**: 서브탭 제거, 20호기 확장, 발전량+풍속 듀얼 차트
- **하루전예측 풍속/정비 탭**: TBD 비활성화 및 HTML/JS 코드 제거
- **메인페이지**: 발전 현황 KPI, 7일 실적 추이, 유지보수 알림 제거
- **기상예보**: 강수량/낙뢰 항목 제거
- **SAMPLE 배지**: 전체 페이지에서 제거

### Fixed
- **Leaflet 지도 중복 초기화 에러**: `_leaflet_id` 체크로 방지
- **숫자 시작 ID querySelector 에러**: `getElementById`로 변경
- **삭제된 HTML 요소 JS 참조 에러**: todayRainfall/todayLightning 등 제거

## [1.4.0] - 2026-03-18

### Added
- **장기예측 박스플롯 체크박스 토글**: 풍속/기온/파고 박스플롯을 체크박스로 선택 표시 (기본: 풍속만)
- **중기예측 발전소 탭 개선**: 장기예측과 동일한 스타일로 통일
  - 기상 예측 카드를 발전량 예측 위로 이동
  - 풍속/기온/파고 체크박스 토글 (기본: 풍속만)
  - 24시간 O&M 모달 → 인라인 패널로 변경
  - 발전량 예측 별도 카드 분리
- **장기예측 모달 주간 상세 테이블**:
  - 박스플롯 클릭 시 해당 주차 7일간 일별 상세 기상 테이블 표시
  - O&M 가능 여부 판단 기준: 풍속 ≤ 10m/s, 최고기온 ≤ 30°C, 최저기온 ≥ 5°C, 파고 ≤ 1.5m
  - 종합 상태 색상 바, 신뢰도, 원형 인디케이터 (녹색=가능, 빨강=불가)
  - 세부 수치는 마우스 Tooltip으로 표시
- **모달 타이틀 날짜 범위 표시**: `N주차 상세 기상 정보 (M월 D일 ~ M월 D일)` 형식
- **중기예측 기상 예측 (14일)**:
  - 풍속/기온/파고 14일 박스플롯 차트 (별도 카드)
  - 일자 클릭 시 24시간 O&M 가능 여부 판단 모달
  - 시간별 풍속 예측 라인 차트 (단기예측 스타일: 구간별 포인트 색상, 기준선 그리드)
  - O&M 기준: 풍속 ≤ 10m/s, 기온 ≤ 30°C / ≥ 5°C, 파고 ≤ 1.5m

### Changed
- **장기예측 기상/발전량 카드 분리**: 기상 예측과 발전량 예측을 별도 카드로 분리
- **터빈별 히트맵 클릭 방식 변경**: 셀 전체 클릭 → 터빈 라벨만 클릭 시 상세 이동 (파란색+화살표 표시)
- **터빈 상세 페이지 차트 순서**: 풍속 예측 → 발전량 순서로 변경
- **모달 크기 확대**: `max-w-4xl` → `max-w-6xl` (장기), `max-w-[95vw]` (중기)
- **중기예측 레이아웃 개선**: 주평균(1/4) + 일평균(3/4) 한 줄 그리드 레이아웃
- **중기예측 주평균 바 차트 제거**: KPI 카드만 표시하여 공간 효율화
- **중기예측 기온 범위 현실화**: -5~30°C → 5~25°C
- **도넛 차트 범례 방식 변경**: Chart.js 내장 범례 → HTML 범례로 대체
  - 범례 줄 수 차이로 인한 도넛 크기 불일치 문제 해결
  - 3개 도넛 차트 크기 동일하게 통일
- **도넛 차트 Y축 스케일 제거**: `scales: {}` 옵션 적용으로 불필요한 축 비활성화
- **도넛 차트 컨테이너 높이 조정**: `h-[250px]` → `h-[200px]`
- **풍속 구간/색상 통일** (도넛 차트 + 박스플롯):
  - 매우 약풍 (0~3) Sky Blue, 약풍 (3~6) Blue, 중간풍 (6~10) Green
  - 강풍 (10~15) Orange, 매우 강풍 (≥15) Red
- **기온 구간 변경**: 저온 (≤5°C) Sky Blue, 적정 (5~30°C) Green, 고온 (≥30°C) Red
- **코드 구조 개선**: `generateBoxplotData()` 함수 글로벌 스코프로 이동
- **O&M 종합 상태 판단 기준 개선** (장기/중기 모달):
  - 🟢 초록: 풍속 + 파고 + 기온 모두 충족
  - 🟡 노랑: 풍속 + 파고 충족, 기온 미충족 (조건부 가능)
  - 🔴 빨강: 풍속 또는 파고 미충족 (O&M 불가)
- **O&M 인디케이터 변경**: 원형 색상(🟢/🔴) → ✓/✗ 텍스트 아이콘 (차트 색상 혼동 방지)
- **파고 박스플롯 "보통" 구간 변경**: 0.8~1.3m → 0.8~1.5m (장기/중기 모두 적용)
- **O&M 녹색 비율 개선**: 풍속/파고 데이터 생성 범위를 O&M 기준 안쪽으로 조정
- **UI 텍스트 정리**:
  - 장기예측 타이틀: '장기(3개월) 발전량 예측' → '장기(3개월) 예측'
  - 중기예측 타이틀: '중기(2주) 발전량 예측' → '중기(2주) 예측'
  - 탭 이름 축약: '발전소 전체 예측' → '발전소', '터빈별 예측' → '터빈별'
  - 발전량 제목: '월평균/주평균/일평균' → '월간 총/주간 총/일일 총 발전량'
  - 중기예측 주평균 범위 텍스트 제거
- **탭 스타일 개선**: 배경색 추가 (비활성: blue-100, 활성: blue-200)
- **중기예측 정비 스케줄 탭**: TBD로 변경 및 비활성화
- **중기예측 터빈별 탭 재설계** (WTG#1~#5 → WTG#1~#20):
  - 14일×20호기 발전량 히트맵 (날짜별 상대 비교, 5단계 팔레트)
  - 터빈별 14일 총 발전량 랭킹 + 스파크라인 (평균 90% 미만 ⚠️ 표시)
  - 히트맵 호버 시 발전량 + 풍속 툴팁
  - 터빈 클릭 → 상세 페이지 (주간 총 발전량 값 + 일간 총 발전량 라인 차트 + 일 평균 풍속 라인 차트)
  - 현실적 데이터 생성: 날짜별 공통 풍황 + 터빈별 성능 차이 + 풍속-발전량 연동

### Removed
- **강수량 항목 제거**: 장기예측 모달 주간 상세 테이블에서 강수량 변수 삭제
- **중기예측 주평균 바 차트 제거**: KPI 카드로 대체
- **중기예측 24시간 O&M 신뢰도 행 제거**

### Technical Details
- **위험 평가 기준**:

## 4. 정비 계획 연동 기능

### 4.1 정비 계획 입력 모달
- 히트맵 셀 클릭 시 정비 계획 입력 팝업
- 입력 항목: 정비 유형(정기점검/부품교체/긴급수리/기타), 시작/종료 시간, 메모
- 터빈명, 날짜, 예상 발전량/풍속 자동 표시
- localStorage 저장, 기존 이력 복원
- 삭제 기능: 기존 이력 있을 때 삭제 버튼 표시

### 4.2 히트맵 정비 마킹
- 정비 입력된 셀: 빨간 테두리(`ring-2 ring-red-400`) + 🔧 배지
- 저장/삭제 시 히트맵 재렌더링

### 4.3 손실 발전량 표시
- 히트맵 합계 옆에 "손실" 열 추가 (빨간색, 터빈별 정비 시간 기반)
- 발전소 탭 KPI: 정비 손실 발전량(빨간), 정비 반영 예상 발전량(초록)
- 정비 내역 요약 목록 (일자, 터빈, 유형, 시간, 손실량)

### 4.4 발전량 차트 정비 반영
- 일일 총 발전량 차트에 "정비 반영 발전량" 빨간 점선 라인 추가
- 정비 계획이 있는 날만 원래 라인과 차이 발생
- 정비 계획 없으면 점선 숨김

### 수정 파일
- `index.html`: 정비 모달 HTML, 손실 KPI, 정비 반영 KPI, 정비 내역 영역
- `scripts.js`: `openMaintenanceModal`, `closeMaintenanceModal`, `saveMaintenancePlan`, `deleteMaintenancePlan`, `updateMaintenanceLoss`, `updateHeatmapMarkers`, `_reRenderHeatmap`

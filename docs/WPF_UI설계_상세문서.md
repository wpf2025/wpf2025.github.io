## 1. 전체 레이아웃 구조

### 1.1 페이지 구성

```
┌──────────────────────────────────────────────────────────┐
│  ┌──────────┐  ┌──────────────────────────────────────┐  │
│  │          │  │  Header                              │  │
│  │          │  ├──────────────────────────────────────┤  │
│  │ Sidebar  │  │                                      │  │
│  │ (w-64)   │  │  Content Area                        │  │
│  │          │  │  (flex-1, overflow-y-auto, p-6)      │  │
│  │          │  │                                      │  │
│  │          │  │                                      │  │
│  └──────────┘  └──────────────────────────────────────┘  │
└──────────────────────────────────────────────────────────┘
```

- 전체: `flex h-screen` — 사이드바 + 메인 영역 수평 배치
- 사이드바: 고정 너비 `w-64` (256px), 다크 배경 `bg-gray-800`
- 메인 영역: `flex-1 flex flex-col overflow-hidden`
  - 헤더: 고정 상단 바 (타이틀 + 알림/언어/사용자)
  - 콘텐츠: 스크롤 가능한 본문 영역

### 1.2 사이드바

| 순서 | 메뉴명 | 아이콘 | data-target | 연결 섹션 ID |
|:---:|--------|--------|-------------|-------------|
| 1 | 서남해해상풍력 | `fa-tachometer-alt` | `overview` | `overview-content` |
| 2 | 하루전예측 | `fa-clock` | `shortterm` | `shortterm-content` |
| 3 | 2주예측 | `fa-calendar-week` | `2week` | `s_2week-content` |
| 4 | 3개월예측 | `fa-calendar-alt` | `longterm` | `longterm-content` |
| 5 | 상세 정보 | `fa-info-circle` | `details` | `details-content` |

- 로고 영역: "O&M을 위한 풍력 예측 서비스" (클릭 시 홈으로 이동)
- 하단: 버전 정보 표시 (`v1.5.0`)
- 활성 메뉴: `bg-blue-600 text-white`
- 호버: `bg-blue-600 text-white` (transition 0.2s)

### 1.3 헤더

```
┌──────────────────────────────────────────────────────┐
│  [현재 페이지 타이틀]          🔔(2)  [EN/KO]  사용자명  │
```

- 좌측: 현재 활성 메뉴명 (`text-xl font-semibold text-gray-700`)
- 우측: 알림 벨 (빨간 배지 `2`), 언어 전환 버튼 (EN↔KO), 사용자명

## 2. 공통 UI 규칙

### 2.1 카드 컴포넌트

```css
.card {
    background-color: white;
    border-radius: 0.5rem;
    box-shadow: 0 4px 6px -1px rgba(0,0,0,0.1);
    padding: 1.5rem;
    margin-bottom: 1.5rem;
```

### 2.5 O&M 종합 상태 판단 기준

| 조건 | 색상 | 의미 |
|------|------|------|
| 풍속 ✓ + 파고 ✓ + 기온 ✓ | 🟢 초록 (`bg-green-500`) | O&M 가능 |
| 풍속 ✓ + 파고 ✓ + 기온 ✗ | 🟡 노랑 (`bg-yellow-500`) | 조건부 가능 |
| 풍속 ✗ 또는 파고 ✗ | 🔴 빨강 (`bg-red-500`) | O&M 불가 |

### 2.6 기상 평가 기준 (WEATHER_THRESHOLDS)

| 변수 | O&M 기준 | 주의 | 위험 |
|------|----------|------|------|
| 풍속 | ≤ 10 m/s | ≥ 13 m/s | ≥ 15 m/s |
| 최고기온 | ≤ 30°C | ≥ 28°C | ≥ 30°C |
| 최저기온 | ≥ 5°C | ≤ 8°C | ≤ 5°C |
| 강수량 | — | ≥ 7mm | ≥ 10mm |
| 파고 | ≤ 1.5m | ≥ 1.5m | ≥ 2m |

## 3. 홈화면 — 서남해해상풍력

### 3.1 개요

발전소의 현재 상태를 한눈에 파악할 수 있는 대시보드 페이지입니다. 오늘의 기상 예보, 시간별 풍속/발전량 차트, 발전소 위치 지도를 제공합니다.

### 3.2 레이아웃

```
┌─────────────────────────────────────────────────────────┐
│  grid: lg:grid-cols-3                                    │
│  ┌──────────────────────────────┐  ┌──────────────────┐  │
│  │  lg:col-span-2               │  │  lg:col-span-1   │  │
│  │                              │  │                  │  │
│  │  ┌────────────────────────┐  │  │  ┌────────────┐  │  │
│  │  │ 오늘 시간별 예상 풍속   │  │  │  │ 오늘       │  │  │
│  │  │ (라인 차트, h-250px)   │  │  │  │ 기상예보   │  │  │
│  │  └────────────────────────┘  │  │  └────────────┘  │  │
│  │                              │  │                  │  │
│  │  ┌────────────────────────┐  │  │  ┌────────────┐  │  │
│  │  │ 오늘 시간별 예상 발전량 │  │  │  │ 발전소     │  │  │
│  │  │ (바 차트, h-250px)     │  │  │  │ 위치 지도  │  │  │
│  │  └────────────────────────┘  │  │  │ (h-300px)  │  │  │
│  │                              │  │  └────────────┘  │  │
│  └──────────────────────────────┘  └──────────────────┘  │
└─────────────────────────────────────────────────────────┘
```

### 3.3 좌측 영역 (2/3)

#### 3.3.1 오늘 시간별 예상 풍속

- 차트 타입: Line (Chart.js)
- X축: 0시~23시 (24개 라벨)
- Y축: 풍속 (m/s), max=20, beginAtZero
- 데이터: 랜덤 생성 (3~15 m/s)
- 스타일:
  - 라인: `borderColor: rgb(59,130,246)`, tension 0.3, fill true
  - 포인트: 풍속 구간별 색상 코딩 (radius 4)
  - 배경: `rgba(59,130,246,0.1)`
- Y축 그리드: 풍속 구간 경계선 (3/6/10/15 m/s) 색상 표시

#### 3.3.2 오늘 시간별 예상 발전량

- 차트 타입: Bar (Chart.js)
- X축: 0시~23시
- Y축: 발전량 (MW), beginAtZero
- 데이터: 랜덤 생성 (2~19.2 MW)
- 스타일: `backgroundColor: rgba(99,102,241,0.6)` (인디고)

### 3.4 우측 영역 (1/3)

#### 3.4.1 오늘 기상예보 카드

```
┌──────────────────────────────────┐
│  [23] 10월 23일 목요일    평균 풍속 │

│  │ 24시간 타임라인 색상 바 (h-20px)││
```

- 날짜 원형 배지: `w-12 h-12 rounded-full bg-blue-500 text-white`

  - 10~13 m/s: `#fcd34d` (주황)
  - ≥ 13 m/s: `#fca5a5` (빨강)

- 마커: WTG #1 ~ #20 (20개 터빈 실좌표)
  - 4열 × 5행 배치 (서남해해상풍력 실제 배치)
  - 클릭 시 팝업: `<b>WTG #N</b>`
- 컨테이너: `h-[300px] rounded-lg border`

## 4. 하루전 예측

### 4.1 개요

내일(D+1)의 시간별 발전량과 풍속을 예측하는 페이지입니다. 발전소 전체 예측과 개별 터빈(WTG #1~#20) 예측을 탭으로 구분합니다.

### 4.2 상단 컨트롤

```
┌──────────────────────────────────────────────────────┐
│  하루전 예측                    ◀ 2026-04-07 ▶ [오늘]  │
├──────────────────────────────────────────────────────┤
│  [발전소]  [터빈별]                                    │
└──────────────────────────────────────────────────────┘
```

- 날짜 네비게이션: `◀` (이전일) / `▶` (다음일) / `오늘` 버튼
  - 날짜 표시: `YYYY-MM-DD` 형식
  - 날짜 변경 시 차트 데이터 재생성
- 메인 탭: `발전소` / `터빈별` (tab-button, setupTabs로 초기화)
- 기본 활성 탭: `발전소`

### 4.3 발전소 탭

#### 4.3.1 레이아웃

```
┌──────────────────────────────────────────────────────┐
│  발전소 전체 예측                                      │
│  ┌──────────────────────────────────────────────────┐│
│  │  시간별 발전량 + 풍속          내일 예상 총 발전량  ││
│  │                               ┌──────────────┐  ││
│  │                               │  4.8 GWh     │  ││
│  │                               └──────────────┘  ││
│  │  ┌──────────────────────────────────────────┐   ││
│  │  │  복합 차트 (Bar + Line, h-350px)          │   ││
│  │  │  좌축: 발전량 (MWh) — 바 차트             │   ││
│  │  │  우축: 풍속 (m/s) — 라인 차트             │   ││
│  │  └──────────────────────────────────────────┘   ││
│  └──────────────────────────────────────────────────┘│
└──────────────────────────────────────────────────────┘
```

#### 4.3.2 복합 차트 상세

- 차트 타입: Bar (발전량) + Line (풍속) 복합
- X축: 0시~23시 (24시간)
- 좌축 (Y): 발전량 (MWh), beginAtZero
- 우축 (Y1): 풍속 (m/s), beginAtZero, grid 비표시
- 데이터셋:
  1. 발전량 (Bar): `rgba(147,197,253,0.7)` (연한 파랑), order 2
  2. 풍속 (Line): `rgb(245,158,11)` (주황), tension 0.3, pointRadius 3, order 1
- 풍속 기본 패턴: 새벽 저풍속(2~3) → 오전 상승 → 낮 최고(10~10.5) → 저녁 하강
- 발전량: 풍속 기반 계산 (컷인 3m/s 미만 → 0.2, 3~6 → w×0.8, 6~10 → w×1.3, 10+ → w×1.4)
- KPI: `bg-blue-50 rounded-lg`, `text-xl font-bold text-blue-700`
- 범례: 하단 표시 (`position: 'bottom'`)

## 7. 인터랙션 흐름 요약

### 7.1 페이지 전환

```
사이드바 메뉴 클릭
  → setActiveSection(targetId)
    → 모든 content-section hidden 처리
    → 대상 섹션만 표시
    → 해당 섹션의 기본 탭 활성화 (발전소 탭)
    → initChartsForCurrentView() 호출
      → destroyAllCharts()
      → 현재 보이는 섹션의 차트만 생성
```

### 7.2 탭 전환

```
탭 버튼 클릭
  → 모든 탭 비활성화
  → 클릭된 탭 활성화 (active 클래스 + aria-selected)
  → 모든 패널 hidden
  → 대상 패널만 표시
  → initChartsForCurrentView() 재호출
```

### 7.3 박스플롯 클릭 → 상세 패널

```
2주 예측: 박스플롯 일자 클릭
  → showMidtermWeatherDetail(dayIndex)
    → 인라인 패널 표시
    → 24시간 풍속 라인 차트 생성
    → O&M 판단 테이블 렌더링
    → 패널로 스크롤

3개월 예측: 박스플롯 주차 클릭
  → showWeatherDetail(weekNumber)
    → 인라인 패널 표시
    → 3개 도넛 차트 생성 (풍속/기온/파고)
    → HTML 범례 + 통계 렌더링
    → 주간 일별 O&M 테이블 렌더링
    → 패널로 스크롤
```

### 7.4 터빈 상세 진입/복귀

```
히트맵 터빈 라벨 클릭 또는 랭킹 항목 클릭
  → showTurbineDetail(t)
    → Overview 영역 hidden
    → 상세 영역 표시
    → KPI 업데이트
    → 풍속/발전량 차트 생성

← Overview로 돌아가기 클릭
  → backToOverview()
    → 상세 영역 hidden
    → Overview 영역 표시
```

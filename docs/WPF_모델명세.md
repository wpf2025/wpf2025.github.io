| 출처 | 모델 | 비고 |
|------|------|------|
| 기존 (하루전 확대) | 중기 기상/발전량 (발전소 1 + 터빈별 20) × 2 | 기존 하루전 모델을 2주로 확대 |
| 펜타 추가 개발 | 발전소 전체 모델, 터빈별 모델 × 20 (후류 포함) | ~4/13 협의 |

|------|------|

### 2.2 I/O

| 방향 | 항목 | 소스 | 비고 |
|------|------|------|------|
| In | NWP 예보 (2주치) | OpenMeteo API (ECMWF, ICON 등) | 기존 하루전과 동일 |
| In | SCADA 데이터 | Dive 경로정보 | 기존 하루전 데이터 사용 |
| In | GFS-WRF 기상 | 공주대 WRF 서버 | 신규 연결 |
| Out | 시간별 평균 풍속 | → Boxplot, O&M 판단 | |
| Out | 시간별 최고/최저 기온 | → Boxplot, O&M 판단 | |
| Out | 시간별 평균 파고 | → Boxplot, O&M 판단 | 공주대 수식 적용 |

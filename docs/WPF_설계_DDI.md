## 2. UI/UX 구현 명세

### 2.1 장기 예측 (3개월) 화면

| 카테고리 | 차트 타입 | 데이터 소스 |
|----------|-----------|-------------|
| 발전량 | 월간 총 발전량 Bar, 주간 총 발전량 Line | 발전량 API |
| 기상 (체크박스 토글) | 주별 풍속/기온/파고 Boxplot | 기상 API (시간 단위) |
| 주차별 상세 기상 | 풍속/기온/파고 분포 Donut | 기상 API (주간 분포) |
| 주간 일별 상세 | O&M 가능 여부 판단 테이블 | 기상 API (일간) |

### 2.2 중기 예측 (2주) — 발전소 화면

| 카테고리 | 차트 타입 | 데이터 소스 |
|----------|-----------|-------------|
| 발전량 | 주간 총 발전량 Text, 일일 총 발전량 Line | 발전량 API |
| 기상 (체크박스 토글) | 일별 풍속/기온/파고 Boxplot | 기상 API (시간 단위) |
| 시간별 상세 기상 | 시간별 풍속 예측 Line | 기상 API (시간 단위) |
| 24시간 상세 | O&M 가능 여부 판단 | 기상 API (시간 단위) |

### 2.3 중기 예측 (2주) — 터빈별 화면

| 카테고리 | 차트 타입 | 데이터 소스 |
|----------|-----------|-------------|
| Overview 히트맵 | 14일 × 20호기 일 발전량 상대비교 | 터빈별 발전량 API |
| Overview 랭킹 | 14일 총 발전량 순위 가로바 + 스파크라인 | 터빈별 발전량 API |
| 상세 페이지 | 주차별 총 발전량 Text, 일간 발전량/풍속 Line | 터빈별 발전량+기상 API |

히트맵 상세:
- 날짜별 일 발전량 상대 비교 (5단계 팔레트)
- 툴팁: 일 총 발전량 + 일 평균 풍속
- 터빈 라벨 클릭 → 상세 페이지 이동
- ⚠️ 평균 대비 90% 미만 터빈 플래그 표시

### 2.4 공통 UI 규칙

**O&M 종합 상태 표시**

| 조건 | 색상 | 의미 |
|------|------|------|
| 풍속 ✓ + 파고 ✓ + 기온 ✓ | 🟢 초록 | O&M 가능 |
| 풍속 ✓ + 파고 ✓ + 기온 ✗ | 🟡 노랑 | 조건부 가능 |
| 풍속 ✗ 또는 파고 ✗ | 🔴 빨강 | O&M 불가 |

**기상 변수 색상 코딩**

- 풍속 5구간: 매우약풍(0~3) SkyBlue, 약풍(3~6) Blue, 중간풍(6~10) Green, 강풍(10~15) Orange, 매우강풍(≥15) Red
- 기온 3구간: 저온(≤5°C) SkyBlue, 적정(5~30°C) Green, 고온(≥30°C) Red

## 3. API 설계

### 3.1 필요 API 목록

| API | 시간 단위 | 데이터 항목 | 사용 화면 |
|-----|-----------|-------------|-----------|
| 장기 기상 | 시간 | 풍속, 기온, 파고 | Boxplot |
| 장기 기상 | 일간 | 평균풍속, 최고기온, 최저기온, 평균파고 | O&M 판단 |
| 장기 기상 | 주간 | 풍속/기온/파고 분포 | Donut |
| 장기 발전량 | 주간 | 발전량 | Bar/Line |
| 중기 기상 | 시간 | 평균풍속, 최고기온, 최저기온, 평균파고 | Boxplot, O&M |
| 중기 발전량 | 일간 | 총 발전량 | Line |
| 중기 터빈별 기상 | 일간 | 터빈별 평균 풍속 | 히트맵, 상세 |
| 중기 터빈별 발전량 | 일간 | 터빈별 총 발전량 | 히트맵, 랭킹, 상세 |

### 3.2 API 출력 형식

모든 예측 결과는 REST API로 출력 (기상예측 + 발전량 통합)

## 4. 데이터 연계

### 4.1 장기 예측

| 방향 | 연계 대상 | 방식 |
|------|-----------|------|
| In | WRF 서버 (공주대) | 위치정보 기반 기상/발전량 데이터 수집 (기존 SCADA 수집 방식) |
| Out | 프론트엔드 | API 출력 |

### 4.2 중기 예측

| 방향 | 연계 대상 | 방식 | 비고 |
|------|-----------|------|------|
| In | OpenMeteo API | NWP 2주치 (ECMWF, ICON 등) | 기존 하루전과 동일 |
| In | SCADA (Dive) | 경로정보 | 기존 하루전 데이터 사용 |
| In | GFS-WRF (공주대) | 경로정보 | 신규 연결 |
| In | 펜타 모델 결과 | 발전소/터빈별 예측값 | 신규 연동 |
| Out | 프론트엔드 | API 출력 | |

## 2. O&M 가능 여부 판단 기준

### 2.1 개별 변수 기준

| 변수 | 기준 | 초과 시 |
|------|------|---------|
| 풍속 | ≤ 10 m/s | O&M 불가 |
| 최고기온 | ≤ 30°C | 조건부 가능 |
| 최저기온 | ≥ 5°C | 조건부 가능 |
| 파고 | ≤ 1.5m | O&M 불가 |

### 2.2 종합 상태 판단

| 조건 | 색상 | 의미 | 운영 조치 |
|------|------|------|-----------|
| 풍속 ✓ + 파고 ✓ + 기온 ✓ | 🟢 초록 | O&M 가능 | 정비 작업 진행 |
| 풍속 ✓ + 파고 ✓ + 기온 ✗ | 🟡 노랑 | 조건부 가능 | 기온 주의하며 진행 가능 |
| 풍속 ✗ 또는 파고 ✗ | 🔴 빨강 | O&M 불가 | 정비 작업 중단/연기 |

> **검토 요청**: 위 기준값(풍속 10m/s, 파고 1.5m 등)이 서남해 현장 운영 기준과 부합하는지 확인 필요

## 3. O&M 활용 시나리오

### 3.1 장기 예측 (3개월) 활용

| 화면 | 운영 활용 |
|------|-----------|
| 월간 총 발전량 텍스트 KPI + 주간 발전량 Line 차트 | 분기별 발전 계획 수립, 매출 예측 |
| 주별 기상 Boxplot (풍속 고정 + 기온/파고 우축 선택) | 향후 기상 트렌드 파악, 정비 윈도우 사전 계획 |
| 주차별 상세 기상 (Donut) | 풍속/기온/파고 분포 확인, 리스크 사전 평가 |
| 주간 일별 O&M 판단 테이블 | 주 단위 정비 스케줄 수립 |

### 3.2 중기 예측 (2주) — 발전소 활용

| 화면 | 운영 활용 |
|------|-----------|
| 일일 발전량 Line 차트 | 2주 내 발전량 변동 예측, 전력 거래 계획 |
| 일별 기상 Boxplot | 일별 기상 변동성 확인 |
| 시간별 풍속 예측 Line | 정비 작업 시간대 선정 |
| 24시간 O&M 판단 | 내일/모레 정비 가능 시간대 확인 |

### 3.3 중기 예측 (2주) — 터빈별 활용

| 화면 | 운영 활용 |
|------|-----------|
| 14일 히트맵 | 터빈별 발전 성능 한눈에 비교, 이상 터빈 조기 발견 |
| 발전량 랭킹 | 저성능 터빈 우선 점검 대상 선정 |
| ⚠️ 90% 미만 플래그 | 이상 터빈 즉시 인지 |
| 터빈 상세 페이지 | 개별 터빈 발전량/풍속 추이 분석 |

## 4. 초기 화면 (서남해해상풍력) 검토

### 4.1 현재 초기 화면 구성

| 위치 | 화면 요소 | 표시 정보 |
|------|-----------|-----------|
| 좌측 상단 | 오늘 시간별 예상 풍속 차트 | 24시간 풍속 Line (구간별 색상) |
| 좌측 하단 | 오늘 시간별 예상 발전량 차트 | 24시간 발전량 Bar |
| 우측 상단 | 오늘 기상예보 | 평균 풍속, 풍속/기온/파고 범위, 타임라인 색상 바 |
| 우측 하단 | 발전소 위치 | Leaflet 위성지도 + WTG#1~#20 마커 |

### 4.2 WPC 검토 요청 사항

| # | 검토 항목 | 질문 |
|---|-----------|------|
| 1 | 초기 화면 필요 정보 | 현재 표시 정보(풍속/발전량/기상예보/지도) 외에 추가로 필요한 정보가 있는가? |
| 2 | 발전 현황 KPI | 현재 발전량, 금일 누적 발전량, 설비 이용률 등 실시간 KPI가 필요한가? |
| 3 | O&M 상태 요약 | 오늘/내일 O&M 가능 여부 요약 표시가 초기 화면에 필요한가? |
| 4 | 알림/경고 | 이상 터빈, 기상 경고 등 알림 영역이 필요한가? |
| 5 | 발전소 선택 | 다중 발전소(월정, 서남해 등) 운영 시 초기 화면에서 발전소 선택이 필요한가? |
| 6 | 기간 범위 | "오늘" 기준 외에 어제 실적, 주간/월간 누적 등 다른 기간 정보가 필요한가? |
| 7 | 지도 활용 | 터빈 마커에 실시간 상태(가동/정지/정비) 표시가 필요한가? |

## 5. 현장성 검토 항목

| # | 검토 항목 | 질문 |
|---|-----------|------|
| 1 | O&M 기준값 | 풍속 10m/s, 파고 1.5m, 기온 5~30°C가 현장 기준과 일치하는가? |
| 2 | 종합 판단 로직 | 기온 미충족 시 "조건부 가능(🟡)"이 현장에서 유효한가? |
| 3 | 이상 터빈 기준 | 평균 대비 90% 미만이 적절한 임계값인가? |
| 4 | 예측 주기 | 1회/일 추론이 운영 의사결정에 충분한가? |
| 5 | 발전소 구분 | 월정, 서남해 등 다중 발전소 운영 시 필요 기능은? |
| 6 | 이력 관리 | 과거 예측 결과 조회가 필요한가? 보관 기간은? |

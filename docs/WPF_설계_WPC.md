## 2. O&M 가능 여부 판단 기준

### 2.1 개별 변수 기준

| 변수 | 기준 | 초과 시 |
|------|------|---------|
| 풍속 | ≤ 10 m/s | O&M 불가 |
| 최고기온 | ≤ 30°C | 조건부 가능 |
| 최저기온 | ≥ 5°C | 조건부 가능 |
| 파고 | ≤ 1.5m | O&M 불가 |

### 2.2 종합 상태 판단

| 조건 | 색상 | 의미 | 운영 조치 |
|------|------|------|-----------|
| 풍속 ✓ + 파고 ✓ + 기온 ✓ | 🟢 초록 | O&M 가능 | 정비 작업 진행 |
| 풍속 ✓ + 파고 ✓ + 기온 ✗ | 🟡 노랑 | 조건부 가능 | 기온 주의하며 진행 가능 |
| 풍속 ✗ 또는 파고 ✗ | 🔴 빨강 | O&M 불가 | 정비 작업 중단/연기 |

> **검토 요청**: 위 기준값(풍속 10m/s, 파고 1.5m 등)이 서남해 현장 운영 기준과 부합하는지 확인 필요

### 3.1 하루전 예측 (D+1) — 사용자가 보는 화면

VPP 하루전 모델의 결과를 O&M 관점에서 재구성하여 표시합니다. 상단에 날짜 선택(`◀ YYYY-MM-DD ▶ [오늘]`)이 있어 과거/미래 날짜를 조회할 수 있습니다.

#### 발전소 탭

```
┌──────────────────────────────────────────────────────┐
│  시간별 발전량 + 풍속                 내일 예상 총 발전량 │
│                                      ┌────────────┐  │
│  ┌──────────────────────────────┐    │  4.8 GWh   │  │
│  │  Bar: 시간별 발전량 (MWh)     │    └────────────┘  │
│  │  Line: 시간별 풍속 (m/s)      │                    │
│  │  (24시간, 듀얼 Y축)           │                    │
│  └──────────────────────────────┘                    │
└──────────────────────────────────────────────────────┘
```

- 발전량(막대)과 풍속(선)을 하나의 차트에서 동시에 확인
- 풍속이 높은 시간대에 발전량이 비례하는지 직관적으로 파악
- KPI: 내일 예상 총 발전량 (GWh)

**운영 활용:**
- 내일 발전량 규모 사전 파악 → 전력 거래 계획
- 시간대별 풍속 패턴 확인 → 정비 작업 시간대 사전 조율

#### 터빈별 탭

```
┌────────────────────┐  ┌────────────────────┐
│  WTG #1 상세 예측   │  │  WTG #2 상세 예측   │
│  실선: 발전량 (MW)  │  │  실선: 발전량 (MW)  │
│  점선: 풍속 (m/s)   │  │  점선: 풍속 (m/s)   │
└────────────────────┘  └────────────────────┘
│  WTG #3 ...         │  │  WTG #4 ...         │
        ... (총 20개, 2열 × 10행) ...
```

- WTG #1 ~ #20 각각 24시간 발전량 + 풍속 듀얼 차트
- 터빈별 고유 색상으로 구분

**운영 활용:**
- 동일 풍속 조건에서 특정 터빈의 발전량이 낮으면 이상 징후 감지
- 터빈 간 발전 편차 확인 → 점검 우선순위 판단

#### WPC 확인/요청 사항

| # | 항목 | 질문 |
|---|------|------|
| W1-1 | **발전소 탭 추가 정보** | 발전량+풍속 외에 기온/파고 등 추가 표시가 필요한가? |
| W1-2 | **터빈별 이상 감지** | 터빈별 탭에서 이상 터빈을 자동 하이라이트하는 기능이 필요한가? (예: 같은 풍속 대비 발전량 80% 미만) |
| W1-3 | **과거 실적 비교** | 예측값과 실제 발전량을 비교하는 화면이 필요한가? |
| W1-4 | **알림 기능** | 내일 예상 발전량이 특정 기준 이하일 때 알림이 필요한가? |

### 3.2 2주 예측 — 발전소 탭

#### 기상 예측 영역

```
┌──────────────────────────────────────────────────────┐
│  기상 예측 (14일)                                     │
│  [✓풍속] [□기온] [□파고]  ← 체크박스로 변수 선택       │
│                                                      │
│  ┌──────────────────────────────────────────────────┐│
│  │  14일 풍속 박스플롯 (일별 min/Q1/중앙값/Q3/max)    ││
│  │  색상: 풍속 구간별 (SkyBlue~Red)                  ││
│  │  ★ 일자 클릭 시 아래에 24시간 상세 패널 펼침        ││
│  └──────────────────────────────────────────────────┘│
│                                                      │
│  [인라인 상세 패널 — 클릭한 일자의 24시간 상세]          │
│  ┌──────────────────────────────────────────────────┐│
│  │  시간별 풍속 라인 차트 (0시~23시)                   ││
│  │  24시간 O&M 가능 여부 판단 테이블:                  ││
│  │  ┌──────┬────┬────┬────┬─────┬────┐              ││
│  │  │ 항목  │ 0시│ 1시│ 2시│ ... │23시│              ││
│  │  │ 종합  │ 🟢 │ 🟢 │ 🟡 │ ... │ 🔴 │              ││
│  │  │ 풍속  │ ✓  │ ✓  │ ✓  │ ... │ ✗  │              ││
│  │  │ 기온  │ ✓  │ ✓  │ ✗  │ ... │ ✓  │              ││
│  │  │ 파고  │ ✓  │ ✓  │ ✓  │ ... │ ✗  │              ││
│  │  └──────┴────┴────┴────┴─────┴────┘              ││
│  └──────────────────────────────────────────────────┘│
└──────────────────────────────────────────────────────┘
```

- 박스플롯: 14일간 일별 기상 변동 범위를 한눈에 파악
- 체크박스: 풍속(기본)/기온/파고 중 필요한 변수만 선택 표시
- 인라인 패널: 특정 일자 클릭 시 24시간 시간별 O&M 가능 여부 확인
- O&M 판단: 종합 상태(🟢🟡🔴) + 개별 변수(✓/✗), 호버 시 수치 표시

**운영 활용:**
- 박스플롯으로 2주간 기상 트렌드 파악 → 정비 가능 주간 사전 선정
- 특정 일자 클릭 → 시간대별 O&M 가능 여부 확인 → 정비 작업 시간 확정
- 강풍/고파고 예상일 사전 파악 → 안전 대비

#### 발전량 예측 영역

```
┌──────────────────────────────────────────────────────┐
│  ┌──────────┐  ┌──────────────────────────────────┐  │
│  │ 1주차     │  │ ⚠️ 주의: 일평균 예측은 변동성이    │  │
│  │ 7.0 GWh  │  │ 크므로 주간 경향성 파악 용도로     │  │
│  │          │  │ 활용하십시오.                      │  │
│  │ 2주차     │  │                                  │  │
│  │ 7.5 GWh  │  │  일일 총 발전량 라인 차트 (14일)   │  │
│  └──────────┘  └──────────────────────────────────┘  │
└──────────────────────────────────────────────────────┘
```

- 좌측: 주간 총 발전량 KPI (1주차/2주차)
- 우측: 14일 일별 발전량 라인 차트 + 변동성 경고 배너

**운영 활용:**
- 주간 총 발전량으로 2주 발전 계획 수립
- 일별 추이로 발전량 급감 예상일 파악

#### WPC 확인/요청 사항

| # | 항목 | 질문 |
|---|------|------|
| W2-1 | **O&M 판단 기준값** | 풍속 ≤10m/s, 파고 ≤1.5m, 기온 5~30°C가 서남해 현장 기준과 일치하는가? 계절별로 다른 기준이 필요한가? |
| W2-2 | **기상 변수 우선순위** | 풍속/기온/파고 외에 O&M 판단에 필요한 추가 변수가 있는가? (예: 강수량, 시정, 낙뢰) |
| W2-3 | **24시간 상세 활용** | 시간별 O&M 판단 테이블이 실제 정비 계획 수립에 유용한가? 표시 형식(✓/✗)이 적절한가? |
| W2-4 | **발전량 경고 기준** | 일일 발전량이 특정 값 이하로 예측될 때 경고 표시가 필요한가? 기준값은? |
| W2-5 | **정비 윈도우 표시** | 연속 🟢 시간대를 "정비 가능 윈도우"로 자동 표시하는 기능이 필요한가? |

### 3.3 2주 예측 — 터빈별 탭

#### Overview 화면

```
┌──────────────────────────────────────────────────────┐
│  14일 × 20호기 발전량 히트맵                           │
│  [낮음 ■■■■■ 높음] (날짜별 상대 비교)                  │
│  ┌──────┬────┬────┬────┬─────┬────┬──────┐          │
│  │ 터빈  │D+1 │D+2 │D+3 │ ... │D+14│ 합계 │          │
│  │ WTG#1│ ■  │ ■  │ ■  │ ... │ ■  │ 1234 │          │
│  │ WTG#2│ ■  │ ■  │ ■  │ ... │ ■  │ 1156 │          │
│  │ ...  │    │    │    │     │    │      │          │
│  │ WTG#20│■  │ ■  │ ■  │ ... │ ■  │  987 │ ← ⚠️    │
│  └──────┴────┴────┴────┴─────┴────┴──────┘          │
│  셀 호버: "발전량: N MWh / 풍속: N m/s"               │
│  터빈 클릭 → 상세 페이지 이동                          │
│                                                      │
│  터빈별 14일 총 발전량 랭킹                             │
│  1. WTG#5  ━━━━━━━━━━━━━━━━━━━━  1,450 MWh  ~       │
│  2. WTG#12 ━━━━━━━━━━━━━━━━━━━   1,380 MWh  ~       │
│  ...                                                 │
│  20. WTG#20 ━━━━━━━━━━━━         987 MWh  ⚠️  ~     │
│                                   (⚠️=평균 90% 미만)   │
└──────────────────────────────────────────────────────┘
```

- 히트맵: 20호기 × 14일, 날짜별 상대 비교 5단계 색상
- 랭킹: 총 발전량 내림차순, 스파크라인(14일 추이), ⚠️ 경고
- 터빈 클릭 → 상세 페이지

#### 상세 페이지 (터빈 클릭 시)

```
┌──────────────────────────────────────────────────────┐
│  ← Overview로 돌아가기                                │
│  WTG #N 상세                                         │
│  ┌──────────┐  ┌──────────┐                          │
│  │ 1주차 총  │  │ 2주차 총  │                          │
│  │ 발전량    │  │ 발전량    │                          │
│  └──────────┘  └──────────┘                          │
│  일 평균 풍속 예측 (14일) — 라인 차트                    │
│  일간 총 발전량 (14일) — 라인 차트                      │
└──────────────────────────────────────────────────────┘
```

**운영 활용:**
- 히트맵으로 20호기 전체 성능을 한눈에 비교 → 이상 터빈 조기 발견
- ⚠️ 플래그로 저성능 터빈 즉시 인지 → 우선 점검 대상 선정
- 랭킹 스파크라인으로 발전량 추이 패턴 확인 (지속 하락 vs 일시적)
- 상세 페이지에서 풍속 대비 발전량 관계 분석 → 터빈 이상 원인 추정

#### WPC 확인/요청 사항

| # | 항목 | 질문 |
|---|------|------|
| W3-1 | **이상 터빈 기준** | 평균 대비 90% 미만이 적절한가? 다른 기준(예: 80%, 또는 절대값)이 더 유용한가? |
| W3-2 | **히트맵 비교 기준** | 날짜별 상대 비교(같은 날 터빈 간 비교)가 적절한가? 터빈별 이력 대비 비교가 더 유용한가? |
| W3-3 | **터빈 그룹핑** | 터빈을 위치/열별로 그룹핑하여 표시하는 것이 유용한가? (예: 1열 WTG#1~5, 2열 WTG#6~10 등) |
| W3-4 | **추가 터빈 정보** | 터빈 상세 페이지에 추가로 필요한 정보가 있는가? (예: 가동률, 정비 이력, 에러 코드) |

### 3.4 3개월 예측 — 발전소 전체

```
└──────────────────────────────────────────────────────┘
```

- 통합 박스플롯: 풍속 고정 + 기온/파고 중 하나를 우축에 추가 가능
- 도넛 차트: 풍속 5구간, 기온 3구간, 파고 3구간 분포 비율
- O&M 테이블: 주간 7일 × 종합/신뢰도/풍속/최고기온/최저기온/파고

#### 발전량 예측 영역

```
┌──────────────────────────────────────────────────────┐
│  월간 총 발전량                                       │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐             │
│  │ 1개월차   │ │ 2개월차   │ │ 3개월차   │             │
│  │ 30 GWh   │ │ 32 GWh   │ │ 28 GWh   │             │
│  └──────────┘ └──────────┘ └──────────┘             │
│                                                      │
│  주간 총 발전량 (12주) — 라인 차트                      │
└──────────────────────────────────────────────────────┘
```

**운영 활용:**
- 12주 박스플롯으로 장기 기상 트렌드 파악 → 분기 정비 계획 수립
- 도넛 차트로 풍속/기온/파고 분포 비율 확인 → 리스크 사전 평가
- 주간 O&M 테이블로 정비 가능 주간 사전 선정
- 신뢰도 표시로 예측 불확실성 인지 → 의사결정 시 참고
- 월간/주간 발전량으로 분기 발전 계획 및 매출 예측

#### WPC 확인/요청 사항

| # | 항목 | 질문 |
|---|------|------|
| W4-1 | **3개월 예측 활용도** | 3개월 예측의 주요 활용 목적은? (분기 정비 계획? 매출 예측? 자재 발주?) |
| W4-2 | **신뢰도 허용 범위** | 예측 신뢰도가 몇 % 이상이어야 의사결정에 활용 가능한가? |
| W4-3 | **도넛 차트 구간** | 풍속 5구간(0~3/3~6/6~10/10~15/15+), 기온 3구간(≤5/5~30/≥30), 파고 3구간(<0.8/0.8~1.5/>1.5)이 현장 관점에서 적절한가? |
| W4-4 | **장기 O&M 기준** | 3개월 예측의 O&M 판단 기준이 2주 예측과 동일해야 하는가? 장기는 더 보수적인 기준이 필요한가? |
| W4-5 | **계절별 차이** | 여름/겨울에 따라 O&M 기준값이 달라져야 하는가? (예: 겨울 최저기온 기준 완화) |

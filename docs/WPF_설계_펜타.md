# WPF 예측시스템 — 펜타 설계 문서

> 작성일: 2026-03-19  
> 담당: 펜타  
> 역할: 중기(2주) 발전소/터빈별 기상 및 발전량 예측모델 추가 개발  
> 협의 기한: ~4/13

---

## 1. 역할 범위

| 항목 | 내용 |
|------|------|
| 발전소 전체 모델 | 중기 2주 기상/발전량 예측모델 추가 개발 |
| 터빈별 모델 × 20 | 기상 모델 (물리모델(후류) + 데이터모델 기반 등등 예측), 발전량 모델 |
| 학습/추론 | WPF 서버 업로드 용 패키징 |

---

## 2. 중기 예측 — 발전소 전체 모델

### 2.1 모델 구성

| 모델 | 설명 | 비고 |
|------|------|------|
| 기상 (데이터모델) | 기존 하루전 모델 → 2주 확대 | 기존 모델 확장 (두산) |
| 발전량 (데이터모델) | 기존 하루전 모델 → 2주 확대 | 기존 모델 확장 (두산) |
| 기상 (펜타 모델) | **추가 개발 필요** | **펜타** |
| 발전량 (펜타 모델) | **추가 개발 필요** | **펜타** |

### 2.2 모델 사양

| 항목 | 사양 |
|------|------|
| 예측 범위 | 2주 (14일) |
| 시간 해상도 | 1시간 간격 |
| 학습 주기 | 1회/월 |k
| 추론 주기 | 1회/일 |

### 2.3 출력 데이터

| 시간 단위 | 데이터 항목 |
|-----------|-------------|
| 시간 | 풍속, 기온, 발전량 |


---

## 3. 중기 예측 — 터빈별 모델

### 3.1 모델 구성

| 모델 | 수량 | 설명 | 비고 |
|------|------|------|------|
| 기상 (데이터모델) | × 20 | 기존 하루전 모델 → 2주 확대 | (두산) |
| 발전량 (데이터모델) | × 20 | 기존 하루전 모델 → 2주 확대 | (두산) |
| 기상 (펜타 모델) | × 20 | **물리모델(후류) + 데이터모델** | **펜타** |
| 발전량 (펜타 모델) | × 20 | **추가 개발 필요** | **펜타** |

### 3.2 출력 데이터

| 시간 단위 | 데이터 항목 |
|-----------|-------------|
| 시간 | 풍속, 기온, 발전량 |

---

## 4. 입력 데이터

| 데이터 소스 | 내용 | 비고 |
|-------------|------|------|
| NWP 예보 | OpenMeteo API 2주치 (ECMWF, ICON 등) | 기존 하루전과 동일 |
| SCADA | Dive 경로정보 | 기존 하루전 데이터 사용 |
| GFS-WRF | 공주대 WRF 결과 | 신규 연결 |

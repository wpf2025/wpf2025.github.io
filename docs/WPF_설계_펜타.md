# WPF 예측시스템 — 펜타 설계 문서

> 작성일: 2026-03-19  
> 담당: 펜타  
> 역할: 중기(2주) 발전소/터빈별 기상 및 발전량 예측모델 추가 개발  
> 협의 기한: ~4/13

---

## 1. 역할 범위

| 항목 | 내용 |
|------|------|
| 발전소 전체 모델 | 중기 2주 기상/발전량 예측모델 추가 개발 |
| 터빈별 모델 × 20 | 물리모델(후류) + 데이터모델 기반 터빈별 예측 |
| 학습/추론 | WPF 서버에서 학습 및 추론 수행 |

---

## 2. 중기 예측 — 발전소 전체 모델

### 2.1 모델 구성

| 모델 | 설명 | 비고 |
|------|------|------|
| 기상 (데이터모델) | 기존 하루전 모델 → 2주 확대 | 기존 모델 확장 |
| 발전량 (데이터모델) | 기존 하루전 모델 → 2주 확대 | 기존 모델 확장 |
| 발전소 전체 모델 | **추가 개발 필요** | 펜타 담당 |

### 2.2 모델 사양

| 항목 | 사양 |
|------|------|
| 예측 범위 | 2주 (14일) |
| 시간 해상도 | 1시간 간격 |
| 학습 주기 | 1회/월 |
| 추론 주기 | 1회/일 |

### 2.3 출력 데이터

| 시간 단위 | 데이터 항목 | UI 용도 |
|-----------|-------------|---------|
| 시간 | 평균풍속, 최고기온, 최저기온, 평균파고 | Boxplot, O&M 판단 |
| 일간 | 총 발전량 | Line Chart |

---

## 3. 중기 예측 — 터빈별 모델

### 3.1 모델 구성

| 모델 | 수량 | 설명 |
|------|------|------|
| 기상 (데이터모델) | × 20 | 기존 하루전 모델 → 2주 확대 |
| 발전량 (데이터모델) | × 20 | 기존 하루전 모델 → 2주 확대 |
| 터빈별 모델 | × 20 | **물리모델(후류) + 데이터모델, 추가 개발 필요** |

### 3.2 출력 데이터

| 시간 단위 | 데이터 항목 | UI 용도 |
|-----------|-------------|---------|
| 일간 | 터빈별 평균 풍속 | 히트맵 툴팁, 상세 Line |
| 일간 | 터빈별 총 발전량 | 히트맵, 랭킹, 상세 Line |

### 3.3 이상 터빈 감지

- ⚠️ 평균 대비 90% 미만 발전량 터빈 플래그 제공

---

## 4. 입력 데이터 (DDI에서 제공)

| 데이터 소스 | 내용 | 비고 |
|-------------|------|------|
| NWP 예보 | OpenMeteo API 2주치 (ECMWF, ICON 등) | 기존 하루전과 동일 |
| SCADA | Dive 경로정보 | 기존 하루전 데이터 사용 |
| GFS-WRF | 공주대 WRF 결과 | 신규 연결 |
